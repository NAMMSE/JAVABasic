# 프로그래밍이란?
 - 프로그래밍 언어를 사용해 프로그램을 만드는 것
 - 프로그래밍 언어를 통해 만든 결과물을 소스코드라고 부른다

# 프로그래밍 언어란?
 - 컴퓨터는 0, 1 밖에 모름
 - 0과 1로만 프로그램을 만들기 어렵기 때문에 언어 등장
 - 사람이 알아듣기 편한 영어와 컴퓨터가 알아듣기 편한 0과 1의 중간지점
 - 소스코드를 0과 1로 번역하는 과정을 컴파일이라 한다
 - 컴퓨터가 해석하기 편할수록 low level 언어(c, c++)
 - 작업하기에 편한 언어는 고급언어(java, python)
 - 저급 언어는 개발 속도가 느리고 성능이 뛰어남
 - 고급 언어는 개발 속도가 빠르고 성능이 상대적으로 떨어짐
 
# 컴파일러
 - 작성한 소스코드를 컴파일하는 프로그램
 - 컴파일 진행 결과로 실행파일 생성
 - 작성된 코드를 한번에 번역
 - 컴파일러를 사용하는 언어를 컴파일 언어라고 한다(c, c++, java...)
 
# 인터프리터
 - 코드를 한 줄씩 해석하여 실행
 - 인터프리터를 사용하는 언어를 스크립트 언어라고 한다(python, javascript,...)

# jdk(java development kit, 자바 개발 키트)
 - 자바로 프로그램을 개발하기 위해 필요한 도구들의 모음
 - 자바 컴파일러(javac), 자바 실행환경(jre) 등이 포함되어있다

# jre(java runtime environment, 자바 실행 환경)
 - 자바로 만든 프로그램을 실행하기 위해 필요한 프로그램
 - jdk를 설치하면 포함되어 있지만, 따로 jre만 설치 가능

# ide (integrated edvelopment envionment, 통합 개발 환경)
 - 프로그램을 개발할 때 필요한 프로그램들을 하나로 묶어놓은 프로그램
 - java의 ide : eclipse, intellij, ...
 - c, c++ : vs
